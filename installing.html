Wake The Language

Wake is a fast, expressive, typesafe language that gives you testability from the ground up.

Installing

On Mac OS X

Mac is the easiest place to begin wake developmment. Simply install node, brew, and xcode, and then run brew install wake.

Clone the wake project seed and follow the other steps in the getting started guide once that is successful.

On Linux

Install boost, flex, and bison developer libraries compiled with C++11 ABIs, and then clone the wake source repository. Using gcc-4.8 or newer (maaaybe 4.7), run make bin/wake TEST=false and then sudo make install. Of course, you'll need node to run the code.

Any help making packages for ubuntu, centos, arch, etc would be appreciated!
